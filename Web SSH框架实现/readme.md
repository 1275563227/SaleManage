# sale-manage

## 使用说明：

1、目录说明：
code.zip -> 源码包

sale.war -> 项目导出的war包（建议Tomcat7运行）

sale -> 源码包解压后目录

sale.sql -> 数据库脚本

sale.pdm -> 数据模型

课程设计文档.docx -> 设计说明书


2、采用myeclipse2016CI开发，如果低版本的编译器导入不成功，请谅解。

3、请在数据库中创建名为“sale”的数据库，运行项目Hibernate会自动建表。

4、运行sale.sql脚本，会灌入数据。
